# 📌 Bleach RPG

Gioco di ruolo a turni a tema **Bleach** sviluppato in Java 25 con JavaFX 25.  
Il giocatore sceglie una classe tra Shinigami, Quincy e Arrancar, esplora quattro atti narrativi (Karakura, Seireitei, Hueco Mundo, Convergenza) e affronta nemici in combattimenti a turni con abilità speciali, trasformazioni, oggetti ed equipaggiamento.

---

## 🚀 Come eseguire il progetto

### Prerequisiti
- Java 25 (LTS)
- Gradle — **non serve installarlo**, il wrapper `gradlew` è già incluso nel repository

### Istruzioni

```bash
git clone <url-del-repository>
cd ProgettoMetodologie
```

### Build del progetto
```bash
./gradlew build
```

### Esecuzione
```bash
./gradlew run
```
